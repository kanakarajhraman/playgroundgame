# Playground Energy Explorer (Singapore P5 Science)

An interactive educational web app designed for Primary 5 students to explore energy conversions in a playground setting.

## Features
- Interactive simulations for 5 playground components.
- Real-time energy meters (GPE, KE, Heat/Sound).
- Scored quizzes with immediate feedback.
- Progress tracking using browser local storage.

## Deployment to GitHub Pages
1. Create a new repository on GitHub.
2. Upload `index.html`, `style.css`, and `script.js` to the main branch.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set the source to "Deploy from a branch" and select the "main" branch.
5. Your site will be live at `https://<your-username>.github.io/<repository-name>/`.

## Technical Details
- Built with HTML5, CSS3, and Vanilla JavaScript.
- No external dependencies or CDNs.
- Uses SVG for scalable, lightweight graphics.
